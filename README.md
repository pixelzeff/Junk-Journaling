# Junk Journaling

A beautiful web application for creating and managing digital junk journals with Faye and Zephyr.

## Features

- **Image Upload**: Add multiple images to your digital journal
- **Responsive Gallery**: Beautiful grid layout that works on all devices
- **Modern Design**: Clean, professional interface with smooth animations
- **File Validation**: Only accepts valid image files

## Getting Started

1. Open the `Studio` folder
2. Start a local server:
   ```bash
   python3 -m http.server 3000
   ```
3. Open `http://localhost:3000` in your browser

## Usage

1. Click "Add Images" to select image files from your computer
2. Images will appear in the gallery below
3. Hover over images to see zoom effects

## Technologies Used

- HTML5
- CSS3 (with modern features like CSS Grid and Flexbox)
- Vanilla JavaScript
